if you are able to stop your breath for atlest 20 - 30 sec easily without coughing - then you are not infected.

if with in 5 sec you are coughing - then you need to consult doctor.

if oxygen level is below 90 - test it with oxymeter - effected.

(1) dry cough + sneeze = air pollution
(2) cough + mucus + sneeze + runny nose = common cold
(3) Cough + mucus + sneeze + runny nose + body ache + weakness + light fever = flu
(4) Dry cough + sneeze + body pain + weakness + high fever + difficulty breathing = coronavirus
